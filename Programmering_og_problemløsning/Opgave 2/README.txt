##8i0.fsx

Koden køres ved at compile 8i0.fsx med img_util.dll biblioteket:

$  cd src
$  fsharpc -r img_util.dll 8i0.fsx

##XML-dokumentation
XML-dokumentationen laves med følgende kommando:

$  cd src
$  fsharpc --doc:documentation.xml -r img_util.dll 8i0.fsx
